Getting Started
--------------------------
Get rcm:

.. code-block:: bash

     sudo apt-add-repository ppa:martin-frost/thoughtbot-rcm
     sudo apt-get update
     sudo apt-get install rcm

Then run:

.. code-block:: bash

    ./install.sh


Gnome Terminal
--------------------------
To get gnome-terminal with a solarized color scheme go into the
gnome-terminal-solarized folder and right ./set_light.sh
